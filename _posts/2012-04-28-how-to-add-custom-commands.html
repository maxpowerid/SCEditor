Code behind it

To add the HTML command, use the $.sceditor.setCommand function:

{% highlight javascript %} $.sceditor.setCommand( "headers", function(caller) { var editor = this, content = $("
"), clickFunc = function (e) { editor.execCommand("formatblock", ""); editor.closeDropDown(true); e.preventDefault(); }; for (var i=1; i<= 6; i++) { content.append( $('Heading ' + i + '') .data('sceditor-hsize', i) .click(clickFunc)); } editor.createDropDown(caller, "header-picker", content); }, "Format Headers" ); {% endhighlight %}

Then add the commands CSS:

{% highlight css %} .sceditor-button-headers div { background: url('/images/headers-button.png'); } .sceditor-header-option { display: block; cursor: pointer; font-size: 14px; text-decoration: none; color: #222; } .sceditor-header-option:hover { background: #eee; } {% endhighlight %}

Finally add the command to the toolbar of the editor:

{% highlight javascript %} $("#demo-cust-cmd").sceditorBBCodePlugin({ toolbar: "headers|source", }); {% endhighlight %}

See the documentation for more details on the above functions and CSS.
